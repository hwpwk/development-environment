[ユーザー名とメールアドレスを設定する]
「ユーザー名」
「git config --global user.name "｛ユーザー名｝"」と入力し、「Enter」キー

「メールアドレス」
「git config --global user.name "｛ユーザー名｝"」と入力し、「Enter」キー

設定した値を確認するには、「git config --list」と入力し、「Enter」キー
